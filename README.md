# Ejcutar en desarrollo
1. Clonar el repositorio
2. Ejecutar
 ```
 
 npm i

 ```

3. Tener Nest CLI
```
npm i -g @nestjs/cli
```

4. Levantar la base de datos
```
docker-compose up -d
```

## Stack Usado
 * MongoDB
 * Nest
